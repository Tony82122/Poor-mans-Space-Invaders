Welcome to **Poor Man's Space Invaders** – a classic arcade-style shooter game built with Python and Pygame!

## Game Description

Its not much but honest work

## Controls
- **Arrow Keys**: Move your ship left and right
- **Spacebar**: Fire bullets
- **Q**: Quit the game

## How to Run
1. Make sure you have Python 3.x installed.
2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run the game using the launcher:
   - On Windows: Double-click `launch_game.bat` or run it in a terminal.
   - Or run directly:
   ```
   cd Alien_Invasion
   python Alien_Invasion.py
   ```

## Requirements
- Python 3.x
